# Chat-app-MERN
Simple chatapp with MERN stack using ws for realtime

# Simple setup
1. Setup the .env file for both API and CLIENT
2. Connect to MongoDB
## Testing in localhost
1. API<br>
`cd api`<br>
`node index.js`<br>
Or you can write a script for your own in package.json
2. CLIENT<br>
`cd client`<br>
`npm run dev`<br>
Or you can write a script for your own in package.json
